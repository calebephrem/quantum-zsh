# Contributing to Quantum Theme

Thank you for considering contributing to Quantum. All kinds of contributions are welcome, from bug fixes to new features, documentation improvements, or any suggestions.

## Getting Started

1. Fork the repository
2. Clone your fork

```sh
   git clone https://github.com/your-username/quantum-zsh
   cd quantum-zsh
```

3. Create a branch for your change

```sh
   git checkout -b feature/your-feature-name
```

## Making Changes

- Follow the existing code style and conventions
- Keep commits clear and descriptive
- Test your changes in a fresh shell before submitting

## Committing & Pushing

```sh
git commit -m "<type>[optional scope]: <description>"
git push origin feature/your-feature-name
```

## Opening a Pull Request

- Open a pull request against the `main` branch
- Describe what you changed and why
- Link any related issues if applicable

## Reporting Bugs

Open an issue at https://github.com/calebephrem/quantum-zsh/issues and include:

- Your zsh version (`zsh --version`)
- Your oh-my-zsh version
- What you expected to happen vs what actually happened

## Suggestions & Ideas

Not feeling like coding? You can still help by:

- Reporting bugs
- Suggesting new features or segments
- Improving readme/docs
- Sharing & starring the project :3
